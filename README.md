# Fibonacci in miniSTARK

## Getting set up

```bash
# 1. install rust https://www.rust-lang.org/tools/install
# 2. make sure rust nightly is installed
rustup toolchain install nightly

# 3. run to make sure everything is working
cargo +nightly build
```
